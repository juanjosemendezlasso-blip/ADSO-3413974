## Diapositiva 2

# Problemática

Muchas organizaciones administran las incidencias mediante llamadas telefónicas, correos electrónicos o aplicaciones de mensajería.

Esto genera inconvenientes como:

- Pérdida de información.
- Duplicidad de solicitudes.
- Tiempos elevados de respuesta.
- Dificultad para realizar seguimiento.
- Baja trazabilidad de las incidencias.

---

## Diapositiva 3

# Solución Propuesta

El Sistema de Gestión y Soporte a Incidencias (SGSI) permitirá:

- Registrar incidencias.
- Clasificar cada solicitud.
- Asignar responsables.
- Realizar seguimiento.
- Actualizar estados.
- Cerrar incidencias.
- Generar reportes.

## Diapositiva 6

# Conclusiones

SGSI permitirá mejorar la administración del soporte técnico mediante la centralización de la información y el seguimiento de cada incidencia.

La adaptación del proyecto establece una base para continuar con las siguientes fases de análisis, diseño e implementación del sistema.
